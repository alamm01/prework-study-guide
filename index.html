HTML

  • head element contains information for browser to help render the page
  • body element contains visible content for the user
  • HTML helps create structure with both visible and invisible content
  • Semantic elements should be used for accessbility
  • head, body, main, ul, ol, li, section,...

CSS

  • we can style HTML elements using the elements as selectors or by using the class attribute as a selector.
  • A margin indictes how much space we want around the outside of an element
  • A padding indicates how much speace we want aroudn the content inside an element
